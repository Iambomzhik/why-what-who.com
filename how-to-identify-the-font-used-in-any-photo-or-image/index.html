How to identify the font used in any photo or image

ByAd007

May 23, 2022

How to identify and download a font using Photoshop

In 2015, Adobe introduced a tool in Photoshop called «Font matching«. This tool makes use of Adobe’s OCR recognition engine to analyze any image or any background and be able to find out what font it is.

To do this, once the image is scanned, Photoshop makes use of the TypeKit database to compare the scanner with the recognized fonts and show us a series of suggested fonts. Once we have found the font we were looking for, we will be able to know its name to download it from any page, or take advantage of TypeKit to download, install, and add it directly to Photoshop.

This option is found within Photoshop, in the section «Text > Font Matching«.

We select this tool and what we will have to do next is adjust the box that appears to the text that we want to scan. The more accurate you are, the more accurate the results obtained by the tool will be.

In a few seconds a series of sources similar to the one we have scanned will appear. Photoshop will show us in the list fonts that we have installed on the computer, and if we have signed in to Adobe Creative Cloudit will also show us the ones from the TypeKit database.

We choose the font that we want to download and use and that’s it. Now we can start working with these fonts.

Why do I need to identify sources of texts

We live in a time when technology seems to be marked by everything related to images, graphics, videos or 3D. However, we can never leave aside something as important as the texts that we edit, use and add to all kinds of documents from a multitude of programs. In fact, despite the widespread use of videos, photos and the like, text is still a key element in most of the work we do on a PC. This can even be extrapolated to the design projects that are being talked about so much at the moment.

That is precisely where the different fonts that we are talking about in these lines come into play. The fonts, fonts or typefaces that we can use at the moment number in the thousands. Some better known and used than others, some paid, many others free. And it is precisely in these design projects, especially in the largest ones, where knowing the type of font used becomes more important. This will allow us, for example, to continue working and expanding the projects of others who have used certain sources, without being out of tune. Also, keep in mind that the ways to identify these elements that we are talking about here are very fast and effective.

Moreover, thanks to the websites that we are going to show you below, we will save a good amount of time in this type of specific task, and without having to pay in most cases.

Other websites to identify and download fonts

If we don’t like Adobe’s tool, or don’t have access to Creative Cloud, there are also other free websites that take care of scanning any image or text to tell us what font you’re using.

WhatTheFont

One of these pages is WhatTheFont. This application allows us to upload any image to its server so that, using its OCR recognition engine, it identifies the text of the image and allows us to analyze any typography in search of its identity.

We will be able to see several results similar to the source that we have analyzed. Next to each result we can see its name (to search for it on other sites) and the possibility of buying said font in this store. This will be very useful for us, since, from this same website, once we access it, we can carry out various tasks related to these sources that we mentioned.

WhatFontIs

Another similar alternative is WhatFontIs. This website also allows us to upload any image from our computer to analyze it and be able to identify the sources it uses. If we register, the OCR will recognize all characters automatically. If not, we will have to introduce some ourselves (we recommend the most characteristic of the source) to be able to compare it.

This website will show us the names of the fonts and the possibility of buying it, if it is paid, or downloading it. We can also filter between free fonts and commercial fonts. We must bear in mind that depending on the type of use of these contents, sometimes it will be profitable for us to acquire paid sources, more so if we are going to use them in professional environments.

Identifont

Identifont is another alternative that will help us identify any typographic font that we can see on the Internet. This website not only allows us to upload a fragment of the font to be able to analyze it and tell us which one it is (or which one is similar to it), but it also allows us to guess which font it is based on a series of questions.

In addition to uploading a photo with the font that we want to guess, we can also identify it by appearance, on the one hand, from the name or by similarity. It even allows us to search for fonts based on their creator. A most complete option to be able to discover new fonts and use them in our projects.

FontFace Ninja

In this specific case we find an option to find sources that is somewhat different from those previously exposed. We tell you all this because when we talk about FontFace Ninjawe are actually referring to an extension for Google Chrome. The main task of this, as you can imagine, is to identify the fonts of the web pages that we visit. To do this, all we have to do is install it in the web browser and click on the extension icon so that it starts working by itself.

In this way, the mouse pointer itself becomes a powerful tool to identify sources of all kinds. This is something that we achieve by passing it over the text that interests us. At that moment we will be shown the typography that is in addition to other data corresponding to it. Of course, we must bear in mind that the extension works with texts as such, not with texts in images. With everything and with it, its operation is very effective with the elements that we commented on.

FontSquirrel Matcherator

Despite the fact that we do not know what the page is about by its name, the truth is that we are facing an important web page, from where we can download a wide range of different sources. From here we will be able to identify an image source with text. It is possible both to load an image from our PC, simply with drag and drop or by providing an image URL, and it is compatible with OneType functions.

It is a fairly precise tool since it will offer us different fonts so that we can choose the one that has the best match with respect to the image that we have added. It is compatible with JPG or PNG formats and we can also manually specify the characters in case the results provided do not convince us. When obtaining the results of the searches we will discover that the fonts have to be paid for.

We can use FontSquirrel Matcherator by accessing its official website.

Fontspring Matcherator

We are talking about a web tool to detect the font used in any image, quite easy to use, since it will only be necessary to click on the font so that it can be identified. Its design is simple as well as attractive and intuitive. We only have to take care of loading the image that contains the font that we want to identify or use a URL link to the image. In a matter of a few seconds, the page will compare the typographic forms and we will obtain the most similar results.

Its main drawback is that, if we need to download the fonts, they are quite expensive, so it is a recommended tool only if we want to know the name of the font. If we want to use Fontspring Matcherator to detect the source of any image, all we have to do is access its web page.

How to use the font in other programs?

We have seen how we can identify the font, or typography, of an image, something very useful if, for example, we are making a photomontage and we do not want to leave evidence. But what if we liked the font and want to use it in other programs, such as Word?

In that case, what we must do is install it in Windows as if it were just another font. We can do this in two different ways, on the one hand, from the Configuration panel, within Personalization > Fonts, and on the other by right-clicking on it and choosing the install option. Be that as it may, we must have previously downloaded the source.

Once installed as one more system font, we can use it in practically any program. Of course, if, for example, we have Word (or whatever program) open, we must close it and reopen it to load the font.

I made the wrong font, what do I do?

As we have seen, programs and font identification tools can sometimes give us several similar but different options. And, although we may think that one of them is quite similar to the font we were looking for, when we download it, install it and start working with it we can find some nuances that show us that it is not.

If we have already installed the font on the computer, what can I do? A font installed on the computer does not mean anything either. As standard, surely in Windows we have installed a few fonts that we have never needed, and will never use again. Some of them were installed by default with Windows, while others may have been installed by programs such as Office, LibreOffice, Acrobat or Photoshop.

The fonts that we don’t need are just there, installed in Windows. But they take up virtually no space on your hard drive, nor do they affect your computer’s performance. Therefore, unnecessary fonts is not something we need to worry about.

If we have installed a font thinking that it was the one we needed, but later it turns out that it was not, we can forget about it and install any other one to see if we are right this time. But, if we are one of those who prefer to have the most minimalist and tidy system possible, we will also be able to uninstall the fonts we don’t want to have in the system from this same configuration panel.
