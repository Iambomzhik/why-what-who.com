Photo, music and video extensions for Mozilla Firefox

We will can see basic information of each photo, such as the location, the ISO, the camera model with which it was captured, the shutter speed, among other parameters. In addition, we can see the images of the last ten open tabs and update the current background image with any of them.

We can download and install Stella Photos from the Firefox extensions website.

Image Picka, download photos in batches

We are talking about a free and open source extension that will allow us to download an image with just drag and drop or by pressing Ctrl/Shift/Alt + click. We will be able to download all the images that we want in batches, either from the current tab or from the tab on the right. It is possible to filter images based on width, minimum or maximum height, file size, image URL, as well as copy URLs of selected images.

We can also give a custom file name, handle invalid characters in the file name, download from the browser cache, and even transform the URL with find and replace. The plugin works well with most websites, though with Instagram we can find some problemsbecause it only shows us white squares instead of the images that we can choose and save.

Download Image Picka from the Firefox Browser Add-ons.

Download Image from Context Menu, download photos from the context menu

By using this plugin we can use the context menu to download images directly without any notice. This, which seems like something so simple, can be very useful since it will allow us to save several photos at the same time. By simply pressing the Enter key after each save, we will avoid going through the annoying “Save As” message.

In order to use this extension correctly, it is important that we do not have the Strict option established in Firefox’s privacy options, since this mode interferes with the extension when downloading images that do not have valid file types. However, this does not affect the content of the file and it can be opened if we provide a valid extension.

If we want to use it, all we have to do is download Download Image from Context Menu from the Firefox website.

Extensions to enjoy music

For music lovers or simply for those who listen to audio from the browser, these extensions for Firefox allow us to improve our experience when playing sound.

Audio Equalizer, configure the audio you hear through the browser

It is a complement that will allow us to easily adjust the audio setup that we want to hear from a toolbar pop-up. In this way we can find the perfect balance between the components of the sound file. For this, we will have several preset settings to be able to choose from a list, such as pop, club, party, rock, among other preset formats.

It is also possible to define our own format to later save it inside the toolbar popup and use it when needed. In addition, it has a reset button through which we will be able to revert all the changes made to the factory settings. For people with hearing problems and who are deaf in one ear, the extension has a Mono functionso you never miss a word or sound when listening to music through headphones.

We can download Audio Equalizer from the Firefox website.

SoundFixer, fix problems with sound volume

This extension has been conceived to solve any sound problem associated with streaming videos, whether listening to music or audio from different websites, with single channel audio, or too loud or too soft. It is compatible with such popular websites as YouTube, Netflix, Twitch or Amazon Prime, if it can work equally on all websites.

With SoundFixer we will be able to sensibly understand any YouTube video that is recorded with a very low tone of voice and we are not able to listen correctly. And it is that Google’s video platform usually only uses 30% of the maximum volume, so with this add-on we can play our favorite music at full volume.

SoundFixer is a free add-on that we can download from the Firefox Browser Add-ons.

600% Sound Volume, increase the volume to the maximum

We are talking about an extension that is characterized by being a volume amplifier as simple as it is reliable. With it we will be able to increase the volume of a song or any audio up to 600% more than its maximum, being able to easily control the volume from any tab. It has a detailed control that allows adjust the volume between 0% and 600% and change it in any tab we play audio with one click.

Of course, we must bear in mind that the complement does not work miracles either, because the more we increase the sound of the audio that we are listening to, the more its quality will decrease, although in general terms it is still good and does not distort too much, even at the highest volumes.

If we want to use the 600% Sound Volume extension, we must know that it is free and we can download its latest version available from the Firefox extensions website.

Add-ons to download videos in Firefox

If we like to watch videos on the Internet with these extensions, we can download them directly to our hard drive to be able to watch them without having to be connected to the Internet.

Video DownloadHelper, extract videos from any website

We are talking about a complete tool to be able to extract videos from different websites and save them to the hard drive. Every time the plugin detects a video embedded in the website that we are visiting, we can download them by clicking on the icon on the toolbar, which will result in the result, so we only have to click.

It is compatible with the main video storage pages such as YouTube, Facebook, Twitch, Vimeo, among others. We just have to be careful not to download videos protected by copyright. In addition to downloading, Video DownloadHelper is also capable of converting audio and video files to different formats, as well as combining audio and video separately into a single file.

We can download Video DownloadHelper from the Firefox extensions website.

Video Downloader professional, save videos up to 4K

This extension for Firefox allows us download and save to our hard drive, any video embedded in a website, being able to support up to 4K resolutions. It is compatible with such popular websites as YouTube and Vimeo, from where it allows us to play videos in any size from the window.

It also allows us to easily add videos to our video list to have quick access to them whenever we want and can play or download them. Of course, this plugin currently does not support the download of videos protected by the RTMP protocol, so we must use them with videos that do not have copyright.

We can download professional Video Downloader from Firefox.

Easy Video Downloader, download audio and video

We are talking about a complement that allows us download both audio and video as it works with its own internal and external download manager like IDM or Wget. We simply have to visit a website and press its button located on the toolbar, so that all the available audio and video sources are listed. We press what we want to download and start downloading it.

It also makes other features available to us, such as the ability to track the information of any image by simply clicking the right button on the toolbar. Also, we can interrupt the download at any time and copy the download links to the clipboard.

Download Easy Video Downloader from the Firefox website.
