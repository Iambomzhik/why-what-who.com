Prevent Windows 11 from spying on you so much with the new version of this tool

ByAd007

May 26, 2022

Although it is obviously not the only operating system that exists, Windows has been the most used in the world for many years. Microsoft’s operating system offers very good features and, above all, great ease of use. However, not all the words that we can say about it are good, and there are many users who have been complaining for years that this collects personal data to send them to company servers. This can be avoided, and that is precisely where it comes in. DoNotSpy.

This espionage of Windows is one of the weak points of the Microsoft system, especially if we take into account the great importance that privacy has gained for users in recent times. The operating system itself allows us to change certain parameters to avoid the collection of some data, but in that sense the tools are somewhat limited. DoNotSpy is a great free software that gives us a lot more options, and that from today we can enjoy in Windows 11.

What is DoNotSpy?

You may be reading these lines and do not know what DoNotSpy is, so we are going to explain what this software is. As we say, Windows itself already gives us some options to adjust the privacy of the system, as well as data collection. Nevertheless, these options are a bit confusing and are somewhat hiddento the point that an average user of the system may not even know of its existence.

This is precisely why DoNotSpy was born. This completely free software launched its version 10 for the previous Windows back in 2015, and in essence it is still a very simple interface for us to adjust dozens of privacy values. The program is really easy to usesince what we will have on the screen will simply be a window that will show us all the available options, and all we have to do is check (or uncheck) those that interest us.

DoNotSpy11 interface (Ghacks)

A couple of preliminary considerations about DoNotSpy. As we say, the tool is completely free although it has advertising. In fact, when we go to install it, it will ask us if we want to install third-party adware, so if we are not attentive we may suffer the “harassment” of these. The second is that despite being a fairly secure program, the truth is that if we don’t use it correctly we may cause a problem in our operating system. Of course, before starting to use it, it will ask us if we want to create a restore pointso we will not have problems in case we do something wrong.

Version 11 is now available

A few hours ago, those responsible for DoNotSpy made available to everyone version 11 of your program. This comes when Windows 11 has been in our homes for a few months, although it is true that there is still a large mass of users who have not dared to make the jump to version. The good thing about this new version 11 is that it uses exactly the same interface and operation as the previous one, so in fact is compatible with both Windows 10 and Windows 11.

By maintaining the interface, we will be able to see few changes in terms of its functionality. DoNotSpy11 will keep showing privacy settings by category, indicating in blue those that are safe to use, in red those that are not recommended for most users, and in orange others that have an associated explanation. The only notable changes in this new version are to be found in some settings specific to Microsoft Officeas well as others related to new features in Windows 11.
