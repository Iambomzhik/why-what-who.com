Update Windows now! New patches fix 20 critical bugs

ByAd007

May 11, 2022

Keeping our operating system updated is key to making this software as secure as possible. It is something that Microsoft already takes care of with Windowsin part thanks to the updates that it sends us monthly correcting the latest vulnerabilities discovered, as is the case.

Right now we are at that time of the month in which the firm sends the latest security patches to the versions of Windows that are still supported. We must bear in mind that our computers increasingly store a greater amount of personal and sensitive data, so we must protect them at all costs. In addition to the antivirus that most of us have installed on the computer, we must keep the operating system and all the software conveniently updated.

We’ve just reached Microsoft’s already popular Update Tuesday where it ships out the latest security patches monthly. Installing them is always important, even more so if we take into consideration that, for example, this month of May 2022 are corrected until 20 critical failures on Windows.

Vulnerabilities fixed by Microsoft in Windows

To give a clear example of all this, we discount, first of all, we will focus on the latest versions of Windows 10 available. We refer to the 20H2, 21H1 and 21H2 in which a total of 44 vulnerabilities are corrected, 4 critical and another 40 important. But that’s not all, but also these security patches update important components of the operating system such as Office.Net VisualStudio.

It is worth mentioning that these versions of the system receive the KB5013942 update that also fixes an issue that shows a black screen for some users when logging in or out. Some solutions for the cloud storage platform are also included. OneDriveor the sudden appearance of News and interests. Also fixes a bug that prevents changing an expired password when logging into Windows.

For their part, Windows 11 users also receive the KB5013943 update to correct the latest vulnerabilities and security flaws detected. Due to all this that we tell you from Microsoft, they encourage us to update the versions of Windows that are still supported as soon as possible. Next, we will help you to carry this out in the fastest and easiest way.

How to update the different versions of the operating system

The first thing we must take into consideration is that, as a general rule, these security patches should reach us automatically through the Windows Update application. To do this, we only have to go to the configuration application through the Win + I key combination. Here it is enough that we go to the section of Update and security to receive these patches automatically. In this way, what we achieve is to be completely protected when using our Microsoft operating system.
